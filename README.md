# 💰 Personal Expense Tracker

A command-line based **Personal Expense Tracker** built using **Python**, **CSV**, and **Matplotlib**. It helps users record daily expenses, analyze spending habits, visualize expenses with charts, monitor monthly budgets, and generate expense reports.

---

## 📌 Features

- ✅ Add daily expenses
- ✅ Store data in CSV format
- ✅ Category-wise expense analysis
- ✅ Monthly expense summary
- ✅ Pie chart visualization
- ✅ Monthly bar chart visualization
- ✅ Budget limit checker
- ✅ Export monthly expense report as a text file

---

## 🛠️ Technologies Used

- Python 3
- CSV Module
- Matplotlib
- OS Module

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/notsneha22/expense-tracker.git
```

### 2. Move into the project folder

```bash
cd expense-tracker
```

### 3. Install Matplotlib

```bash
pip install matplotlib
```

### 4. Run the application

```bash
python expense_tracker.py
```
